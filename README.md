# Rerearch Print ke Thermal Printer dengan React

- Case dengan koneksi usb
- Case dengan socket io
- Cased dengan network - progress

Environment:
- Node: 18.18.2
- pnpm: 9.7.0

Required lib:
- scpos
- escpos-usb
- usb@1.9.2

Start:
- pnpm i
- pnpm add escpos escpos-usb usb@1.9.2
- pnpm transpile:electron
- pnpm dev
